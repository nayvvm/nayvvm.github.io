# Assistente Culinário por Gestos

Um player de vídeos de receita (YouTube) controlado por gestos da sua mão.

**⚠️ AVISO: Este programa funciona apenas em Linux (Ubuntu, Debian, Mint, etc.)**

---

### Como Instalar e Rodar (Para Leigos)

Siga estes 4 passos.

#### Passo 1: Baixar o projeto

Clique no botão verde escrito "<> Code" no topo desta página e depois clique em "**Download ZIP**".



#### Passo 2: Extrair os arquivos

Encontre o arquivo `assitente_culinario.zip` (ou similar) na sua pasta de Downloads. Clique com o botão direito nele e escolha "**Extrair aqui**".

Isso vai criar uma pasta nova. Entre nela.

#### Passo 3: Rodar a Instalação

Você verá vários arquivos. Encontre o arquivo `install.sh`.

1.  Clique com o botão direito nele -> **Propriedades**.
2.  Vá para a aba **Permissões**.
3.  Marque a caixinha que diz "**Permitir execução do arquivo como um programa**".
4.  Feche a janela de propriedades.
5.  Dê **dois cliques** em `install.sh`.
6.  O sistema pode perguntar o que fazer. Escolha "**Executar no terminal**" ou "**Executar**".
7.  Um terminal preto vai aparecer. Ele vai pedir sua senha de usuário (a mesma que você usa para logar no PC). Digite-a e pressione Enter.
8.  Espere o script instalar tudo (pode demorar alguns minutos).

#### Passo 4: Rodar o Programa!

Após a instalação terminar, volte para a pasta e encontre o arquivo `run.sh`.

1.  Assim como no passo 3, dê a ele **permissão de execução** (Botão direito -> Propriedades -> Permissões -> Marcar "Permitir execução...").
2.  Dê **dois cliques** em `run.sh`.
3.  Escolha "**Executar no terminal**" ou "**Executar**".

O programa vai compilar e a janela do Zenity (para pedir a URL do YouTube) deve aparecer!
